# git-cifuentes
Custom Git command that checkout your MASTER branch and pull to get the last changes.

# Installing

    sudo ./install.sh

  OR

    cp git-cifuentes /bin
    chmod a+x git-cifuentes

Then you can execute.

    git cifuentes
